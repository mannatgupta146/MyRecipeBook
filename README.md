# 🍲 MyRecipeBook — Personal Recipe Manager

MyRecipeBook is a modern recipe management web app that lets users discover, create, organize, and save recipes in one beautiful interface.

Built with a focus on simplicity, performance, and real-world usability, this app helps food lovers build their own digital cookbook.

---

## ✨ Features

### 🔍 Discover Recipes
- Explore recipes from around the world
- Fetches real recipes using the Spoonacular API
- Beautiful recipe cards with images & categories

### 📝 Create Your Own Recipes
- Add custom recipes
- Add unlimited ingredients
- Write step-by-step instructions
- Upload recipe images
- Custom categories supported

### ✏️ Edit & Update Recipes
- Modify ingredients, instructions, or details
- Update category or chef name
- Changes saved instantly

### 🗑 Delete Recipes
- Remove recipes permanently
- Confirmation prompt included

### ❤️ Favorites System
- Mark recipes as favorites
- View all favorites in one place
- Favorites auto-remove if recipe is deleted

### 🔎 Smart Search
Search recipes by:
- title
- category
- chef name
- ingredients

### 📱 Recipe Details Experience
- Interactive ingredient checklist
- Step-by-step preparation view
- Share recipe link
- Clean distraction-free layout

### 💾 Local Storage Persistence
- Recipes saved locally
- Data stays after refresh
- User recipes appear first

### 🔄 Restore Default Recipes
- Restore API recipes if deleted accidentally

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/mannatgupta146/MyRecipeBook.git
cd MyRecipeBook
````

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Add Spoonacular API Key

Create a `.env` file in the root:

```
VITE_SPOONACULAR_KEY=your_api_key_here
```

Get your free key from:
👉 [https://spoonacular.com/food-api](https://spoonacular.com/food-api)

### 4️⃣ Run the app

```bash
npm run dev
```
